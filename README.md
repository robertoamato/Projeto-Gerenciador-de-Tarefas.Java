# Gerenciador de Tarefas

Este é um projeto Java simples para gerenciar tarefas via linha de comando. Ele permite adicionar, visualizar, concluir e excluir tarefas, além de salvar o progresso em um arquivo.

## Funcionalidades
- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Exibir todas as tarefas
- Excluir tarefas específicas
- Salvar tarefas em um arquivo de texto

## Como Usar
1. Compile e execute o programa `Main.java`.
2. Siga as instruções no terminal para interagir com o programa.
3. As tarefas são salvas no arquivo `tasks.txt`.

## Estrutura do Projeto
- `Task.java`: Define a classe `Task` que representa uma tarefa.
- `TaskManager.java`: Gerencia uma lista de tarefas.
- `FileHandler.java`: Manipula o salvamento de tarefas em arquivo.
- `Main.java`: O ponto de entrada do programa que lida com a entrada do usuário.

## Tecnologias Usadas
- Java (JDK 8 ou superior)
